
# План: Активировать Игровой зал

## Проблема

Карточка «Игровой зал» в главном салоне помечена как `disabled` с бейджем «Скоро», хотя игровой модуль уже реализован и готов к использованию.

## Решение

Убрать ограничения с карточки `CabinCard` для игрового зала:
- Удалить атрибут `disabled`
- Удалить атрибут `badge="Скоро"`

## Изменяемый файл

| Файл | Изменение |
|------|-----------|
| `src/pages/ethereal/EtherealHome.tsx` | Убрать `disabled` и `badge` у карточки игрового зала |

## Код до изменения (строки 57-65)

```tsx
<CabinCard
  to="/e/games"
  icon={Gamepad2}
  title="Игровой зал"
  subtitle="Игры"
  badge="Скоро"
  disabled
/>
```

## Код после изменения

```tsx
<CabinCard
  to="/e/games"
  icon={Gamepad2}
  title="Игровой зал"
  subtitle="Игры"
/>
```

## Результат

После изменения карточка «Игровой зал» станет активной и будет вести на страницу `/e/games`, где пользователь сможет создать или присоединиться к игровой сессии «Ситуации на борту».
